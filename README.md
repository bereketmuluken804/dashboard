# Admin Dashboard
This is a simple mockup dashboard  from The Odin Project, built with HTML and CSS for exercising CSS grid layout.

# Features
- Grid and Flexbox layout
- Responsive design
- Project, Announcement, and Trending sections

# Technologies
- HTML
- CSS
- CSS Grid
- CSS Flexbox
